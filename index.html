Le projet des voitures

Lien vers la repo github

On projette de lier des voitures grâce à des raspberry, on peut trouver sur la repo le code qui permet d'envoyer des informations en temps réels. Ainsi le code (en python) tourne sur les raspberry et envoie les informations des voitures à un serveur. On peut visualiser toutes les informations envoyées au serveur grâce à une interface graphique. On doit l'idée au magnifique Robin et et le code à l'encore plus magnifique Romain.
